A version of the game "Battleships".

Players can play against the computer, which attacks at random.

Done for practice in the following areas:
* Test Driven Development (TDD) using Jest/Babel
* Bundling (using Webpack)
* Object-Oriented Programming (OOP) working with classes and modules
* Algorithms and game-logic programming

This project is the final project in the JavaScript-course in the [Full Stack JavaScript-path on The Odin Project](https://www.theodinproject.com/paths/full-stack-javascript/courses/javascript).
